# Vulkan Study

本项目是学习 Vulkan 图形 API 的实践项目。

## 项目概述

这是一个基于 C++ 和 Vulkan API 的学习项目，实现了基本的三角形渲染。项目使用 CMake 构建系统，并依赖 GLFW 和 GLM 库。

## 构建说明

项目使用 CMake 和 vcpkg 进行依赖管理：

```bash
# 确保已安装 Vulkan SDK
# 配置 vcpkg 路径
# 使用 CMake 构建项目
```
